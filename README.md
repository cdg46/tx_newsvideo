# Insertion video en texte

## usage

  //VIDEO:PLATEFORME:VIDEOID:Ratio//

  //VIDEO:PLATEFORME:VIDEOID//

TS: config
  plugin.tx_newsvideo {
    ratio = 16by9
  }

ratio est 16/9em ou 4/3

## Plateforme disponibles

  * YOUTUBE
  * VIMEO
  * CLIPFISH
  * KIKAVU
  * DAILYMOTION
  * WAT
  * MEATACAFE

## Responsive

utilisation des class bootstrap 3 pour des videos responsive
